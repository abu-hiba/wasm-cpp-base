# wasm-cpp-base

### Pre-requisite:
Install emsdk by the following the instructions on the
[emscripten site](https://emscripten.org/docs/getting_started/downloads.html)

### To build:
```sh
npm run build
```

### To serve the built files:
```sh
npm run serve
```

### To build & serve in one step:
```sh
npm run start
```
